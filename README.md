# Duino-Coin Dashboard

**Phiên bản:** v1.6 by Razester72

Một công cụ dashboard đơn giản để theo dõi tài khoản Duino-Coin của bạn theo thời gian thực.

## Tính năng

- Hiển thị số dư DUCO thời gian thực
- Danh sách các miner đang kết nối
- Tỷ lệ chấp nhận/reject của từng miner
- Log hoạt động
- Lưu username (Remember me)
- Giao diện tối hiện đại (CustomTkinter)

## Cách sử dụng

1. Chạy file `Duino - Coin DashBoard.exe`
2. Nhập username Duino-Coin của bạn
3. Tick "Remember me" nếu muốn lưu username
4. Chờ khoảng 15 giây để dữ liệu được tải

## Yêu cầu

- Cần có tài khoản Duino-Coin
